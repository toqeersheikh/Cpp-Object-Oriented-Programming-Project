# Comprehensive C++ Mastery Journey 🚀

Welcome to my C++ repository! This project serves as a complete, structured record of my learning journey as I mastered **C++ from foundational syntax to advanced software engineering paradigms**. 

To validate and implement these concepts in a production-like environment, I designed and engineered a full-scale **Banking Application Project** that brings together object-oriented design, robust error handling, and persistent data storage.

---

## 🛠️ Complete Roadmap & Topics Covered

This repository documents hands-on implementation and deep-dives into the following architectural milestones:

### 1. Foundational Systems Programming
* **Core Mechanics:** Data types, variables, type modifiers, and expressions.
* **Control Flow:** Advanced conditional structures (`if-else`, `switch`) and iteration logic (`for`, `while`, `do-while`, range-based loops).
* **Arrays & Matrices:** 1D and multi-dimensional arrays, vector-like behaviors, and matrix manipulations.
* **Pointers & Memory Management:** Pointer arithmetic, dynamic memory allocation (`new`/`delete`), stack vs. heap allocation, and preventing memory leaks.
* **References:** L-value references, passing parameters by reference, and alias mechanics.

### 2. Functional & Modular Architecture
* **Function Design:** Parameter passing methods (by value, by address, by reference).
* **Advanced Functions:** Function overloading, default arguments, inline functions, and recursive problem-solving.
* **Scope Mechanics:** Local, global, static, and block scopes.

### 3. Object-Oriented Programming (OOP) Deep Dive
* **Encapsulation:** Classes, objects, data hiding, access specifiers (`public`, `private`, `protected`), and properties/getters/setters.
* **Lifecycle Management:** Shallow vs. Deep copying, default constructors, parameterized constructors, copy constructors, and destructors.
* **Polymorphism & Inheritance:** * Base and Derived class mechanics, constructor inheritance.
  * Virtual functions, runtime polymorphism, and abstract classes (Pure Virtual Functions).
* **Operator Overloading:** Customizing operators (`+`, `-`, `<<`, `>>`) for user-defined types, using member functions and friend functions.

### 4. Advanced Systems & Exception Safety
* **Templates:** Generic programming with function templates and class templates.
* **Exception Handling:** Robust error boundary architecture using `try`, `catch`, and `throw` blocks, custom exception classes, and stack unwinding.

### 5. Persistent Storage & I/O Streams
* **File Handling:** Utilizing `ifstream`, `ofstream`, and `fstream` to interact with disk storage.
* **Serialization & Deserialization:** Writing complex object states to external files and reconstructing them perfectly back into system memory.
* **Stream Manipulators:** Structuring console and file input/output workflows elegantly.

---

## 🏦 Capstone Project: Automated Banking System

The focal point of this repository is a **Banking Application** that acts as the ultimate integration test for all the concepts detailed above. 

### Key Features Implemented:
* **Account Lifecycle Management:** Open, query, deposit, withdraw, and close bank accounts dynamically.
* **Object-Oriented Architecture:** Designed using clean inheritance and abstract entities to handle diverse account types securely.
* **Data Persistence:** Built an I/O stream engine that automatically saves the entire database of accounts to a physical file upon system exit, and reloads it instantly when restarted.
* **Bulletproof Safety:** Guarded critical user paths (like overdrafts or invalid inputs) using strict exception handling.
